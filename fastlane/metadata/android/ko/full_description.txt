가볍고 배터리 효율적인 GPS Logger(기록) 입니다. 이 앱의 목적은 GPS 좌표를 특정 간격으로 SD카드에 기록하는 것입니다.  이 앱은 당신이 걸을 때나 하이킹할 때, 비행기를 탈 때, 사진을 찍을 때 백그라운드로 동작하며 가능한 오랫동안 실행됩니다.   사진에 지오태그를 달거나, 여행 사이트에 업로드하거나, 구글어스에서 보는 등의 용도로 파일을 이용할 수 있습니다. 

***기능:

* 시간 간격 설정 가능
* 거리 간격 설정 가능
* 네트워크 연결이나 GPS 위성 지원
* GPX와 KML 파일 지원
* 알림 표시
* 장치 시간 혹은 위성 시간 사용
* 야드파운드법(Imperial units) 지원
* 부팅시 자동 시작
* 자동으로 시간마다 이메일 보내기
* OpenStreetMap - GPS 추적 결과 업로드
* Dropbox - GPX/KML/ZIP 형식으로 업로드


버그, 기능 요청, 질문은 github.com/mendhak/gpslogger에 제출해 주세요.  

***공지 사항:

GPSLogger는 MyTrack의 대체본이 아닙니다.  OpenTracks는 많은 기능이 내포되어 있어서 단기적 사용을 목적으로 하지만, GPSLogger는 장기적 사용을 목적으로 합니다.

자동 이메일, OpenStreetMap이나 Dropbox를 사용할 때에만 데이터 연결이 사용됩니다.

이 애플리케이션은 빈도수값 0을 허용하나, 기록이 급격한 속도로 이루어져서 불안정/멈춤현상을 초래할 수 있으므로 권장하지는 않습니다. 대신 1-3초 주기를 적용하세요.

이 앱이 가졌으면 하는 기능이 있다면, GitHub에 요청사항을 보낼 수 있습니다.

***권한 설명:

저장소 - SD카드의 GPSLogger 폴더에 파일 읽기 및 쓰기

네트워크 통신 - Dropbox, Openstreetmap같은 곳에 업로드하거나 이메일 전송, Dropbox, Openstreetmap에 인증 할 때 사용

위치 - GPS 혹은 기지국 기반 위치를 결정하는 데 사용

시스템 도구 (부팅시 자동으로 시작) - GPSLogger를 부팅시에 사용하기를 선택했을 경우 사용
