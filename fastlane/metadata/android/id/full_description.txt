Logger GPS yang ringan dan hemat baterai. Tujuan dari aplikasi ini adalah untuk mencatat koordinat GPS Anda pada interval tertentu ke file di kartu SD Anda. Aplikasi ini berjalan di latar belakang sehingga Anda dapat berjalan-jalan, mendaki, naik pesawat atau sesi foto atau bahkan membeli susu dan menjalankannya selama mungkin. Setelah Anda kembali ke komputer, Anda kemudian dapat menggunakan file tersebut untuk memberi geotag pada foto, mengunggah ke situs perjalanan, melihat di Google Earth, dan seterusnya.


Untuk informasi selengkapnya, lihat <a href="https://gpslogger.app">https://gpslogger.app</a>

<b>Catatan</b>: Ingatlah untuk <b>Izinkan aplikasi ini sepanjang waktu</b> di setelan OS Anda, untuk mengizinkannya mencatat lokasi di latar belakang.

<b>Fitur</b>

<ul>
<li>Dapat masuk ke format GPX, KML, CSV, NMEA, dan CSV. Bisa juga masuk ke URL khusus atau hanya layar.</li>
<li>Mencatat poin GPX sebagai lintasan dan segmen lintasan</li>
<li>Juga catat kecepatan, arah, dan ketinggian jika tersedia</li>
<li>Tambahkan deskripsi ke suatu titik menggunakan menu anotasi, akan ditambahkan sebagai titik jalan</li>
<li>Konfigurasi interval waktu antar titik</li>
<li>Konfigurasi filter jarak minimum antara titik atau filter akurasi</li>
<li>Ikon notifikasi dengan koordinat dan tindakan cepat</li>
<li>Menggunakan menara seluler untuk lokasi saat GPS dinonaktifkan</li>
<li>Pilih jaringan, gps, dan penyedia lokasi pasif secara selektif</li>
<li>Unggah atau email file secara otomatis pada interval yang ditentukan ke tujuan seperti Email, FTP, Dropbox, OpenStreetMap, Google Drive, URL Kustom.</li>
<li>Berbagi lokasi atau file log melalui SMS/Email (dan aplikasi lain seperti Facebook atau Twitter tergantung pada apa yang telah Anda instal)</li>
<li>Mulai masuk saat booting ponsel</li>
<li>Lakukan operasi dari Tasker atau aplikasi otomatisasi lainnya.</li>
<li>Anda dapat memberinya file prasetel dengan nilai Anda sendiri untuk distribusi</li>
<li>Beberapa profil dengan setelan berbeda dimungkinkan</li>
</ul>

Bug, permintaan fitur, pertanyaan - harap kirimkan ke github.com/mendhak/gpslogger (pelacak masalah).

<b>Catatan</b>

Koneksi data digunakan oleh aplikasi ini hanya jika Anda menggunakan fitur kirim otomatis atau fitur URL Kustom

Meskipun aplikasi memungkinkan frekuensi 0 detik, namun tidak disarankan karena perekaman dan pencatatan berlangsung sangat cepat; ini dapat menyebabkan ketidakstabilan/hang. Coba 1-3 detik sebagai gantinya.


<b>Penjelasan izin</b>

Penyimpanan - membaca dan menulis file ke folder GPSLogger di kartu SD Anda

Komunikasi jaringan - digunakan saat mengunggah file (Dropbox, Openstreetmap) atau mengirim email atau mengizinkan Anda menggunakan Dropbox, Openstreetmap

Lokasi Anda - digunakan untuk menentukan lokasi berbasis GPS atau menara Anda

Alat sistem (mulai secara otomatis saat boot) - digunakan jika Anda memilih untuk memulai GPSLogger saat boot

Izin penyimpanan bersama - di Android 11, digunakan untuk mengizinkan penulisan file ke jalur file khusus
